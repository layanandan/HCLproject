How to run the Complaint Management System (CMS) Project

1. Extract the file and copy cms folder

2.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

3. Open PHPMyAdmin (http://localhost/phpmyadmin)

4. Create a database with name cms

5. Import cms.sql file(given inside the zip package in SQL file folder)

6.Run the script http://localhost/cms (frontend)

7. For admin panel

http://localhost/cms/admin  (admin panel)
